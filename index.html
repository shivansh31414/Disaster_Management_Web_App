Stay informed and prepared with real-time weather alerts and emergency checklists for any city worldwide

🎯 Demo Mode - OpenWeaver Design System

This is a demonstration of the Emergency Alert App v2 featuring the OpenWeaver Inspired Design System. The app now includes:

  • 🎨 Modern color palette with purple primary and pink secondary colors
  • 📱 Responsive design with consistent spacing system
  • ✨ Smooth animations and hover effects
  • 🔧 Professional header and footer components
  • 📐 Design system-based typography and spacing

To use with real weather data:

  1. Set up your OpenWeather API key in a .env file
  2. Run npm install to install dependencies
  3. Start the server with node server.js
  4. Access the app at http://localhost:3000/emergency_app.html
